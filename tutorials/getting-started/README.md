# Getting Started with DecentClaude

Welcome to your first tutorial! This guide will walk you through the fundamentals of working with DecentClaude.

## Learning Objectives

By the end of this tutorial, you will:

- Understand the DecentClaude project structure
- Set up your development environment
- Create your first data transformation
- Run data quality checks
- Use Claude Code effectively in your workflow

## Prerequisites

- Basic understanding of SQL
- Familiarity with command line interfaces
- Access to a BigQuery project
- Claude Code installed and configured

## Time Commitment

Approximately 30 minutes

## Tutorial Structure

1. **Step 1**: Understanding the Project Structure (5 min)
2. **Step 2**: Environment Setup and Configuration (10 min)
3. **Step 3**: Your First dbt Model (10 min)
4. **Step 4**: Running Data Quality Checks (5 min)

## Let's Get Started!

Navigate to the `steps/` directory and begin with `01-project-structure.md`.

## Quick Check

Before you begin, verify you have:

- [ ] Claude Code installed
- [ ] Access to BigQuery
- [ ] Python 3.7+ installed
- [ ] Git installed
- [ ] Text editor configured

If you're missing any of these, refer to the [Setup Guide](../../onboarding/setup-wizard.sh).

## Help

Stuck? Common issues and solutions:

- **Environment issues**: Check `onboarding/checklist.md`
- **BigQuery access**: Verify credentials with `gcloud auth list`
- **Python dependencies**: Run `pip install -r requirements.txt` (if exists)
